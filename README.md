# LegacyShield

**Protect Your Family. Build Your Legacy.**

LegacyShield is a financial literacy and estate education platform for DC-area working families. We teach life insurance, estate planning, and generational wealth building — in plain language, from someone in your community.

## About
- **Founder:** Anthony Washington — retired DC law enforcement, Mason, family man
- **Target Market:** Working and middle-class families in the DC metro area
- **Revenue Model:** $39/month subscriptions · $149 workshops · life insurance affiliate referrals

## Tech Stack
- **Frontend:** Next.js 14 + TypeScript + TailwindCSS
- **Deploy:** Netlify

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy

Push to `main` — Netlify auto-deploys.
